Bigdataprocessor2

Ecosystem

BigDataProcessor2 (BDP2) is a Fiji plugin for interactive processing of TB-sized image data. BDP2 uses BigData Viewer for rendering and the ImgLib2 library for image processing The BDP2 is the new version of BiaDataProcessor Main features: lazv-loadina of TB sized image data. lazv-processina of TB sized image data.
